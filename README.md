Getting Started with Development
================================

1. Install node.js
2. Install npm (Node Package Manager)
3. Install mongodb
4. clone the repository
   ```$ git clone git@github.com:lorenyu/abootay-node.git```
5. Install project dependencies:
   ```$ npm install```
6. Run the server
   ```$ npm start```

Running the Server
------------------
```$ npm start```

Stopping the Server
-------------------
```$ npm stop```

Restarting the Server
---------------------
```$ npm restart```

Running Tests
-------------
```$ npm test```
